隱私權政策 (Privacy Policy)

最後更新日期：2026年4月11日

Google API Services User Data Policy Compliance:
本外掛對從 Google API 接收到的資訊的使用與處理，將遵循 Google API 服務使用者資料政策，包括其中的「有限使用」要求。

1. 資訊收集 (Information Collection)

This Add-on does NOT collect, store, or transmit any personal data or document content to external servers. All text processing and phonetic conversions are performed locally within the user's Google Apps Script environment.

本外掛「破音字標注 Poyinzi」不會收集、儲存或傳送任何個人資料或文件內容至外部伺服器。所有的文字掃描、處理與注音轉換均在您個人的 Google Apps Script 環境內本地執行，確保您的文件隱私不外洩。

2. 權限與範圍 (Permissions and Scopes)

The Add-on requests the following permissions only to provide its core functionality:
  • https://www.googleapis.com/auth/documents.currentonly
    用以讀取並修改您目前選取的文字，以便套用注音字體與修正讀音。 (To read and modify the selected text for phonetic correction in the active document.)
  • https://www.googleapis.com/auth/script.container.ui
    用以顯示側邊欄介面，讓您能與工具互動並選擇正確的讀音。 (To display the sidebar for user interaction and phonetic selection.)

3. 第三方服務 (Third-Party Services)

The Add-on uses Google Fonts to render the "Bpmf Zihi Kai Std" font. No data is shared with other third parties.

本外掛僅使用 Google Fonts 服務來載入「字嗨注音標楷體 (Bpmf Zihi Kai Std)」以正確顯示注音。除此之外，我們不會與任何第三方共享您的數據或文件內容。

4. 資料保留 (Data Retention)

Since we do not collect any data, no data retention policy applies.

由於本外掛不具備後端資料庫，且不收集任何使用者資訊，因此不適用資料保留政策。您的資料始終留在您的 Google 文件中。
