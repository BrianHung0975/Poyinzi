# 破音字標注 Poyinzi

解決 Google Docs 無法標注破音字的工具。
Solution tools for fixing the inability to annotate phonetic tone marks (Zhuyin) in Google Docs.

## 專案簡介 Project Overview
這是一款專為教育工作者設計的 Google Docs 擴充功能，旨在解決 Google 文件中無法套用注音字型並智慧校正破音字的問題。

This is a Google Docs Add-on designed for educators, aiming to solve the issue of applying phonetic fonts and intelligently correcting heteronyms in Google Docs.

## 主要功能 Features
- **🎨 一鍵套用注音字型**：快速為選取文字套用注音字型。
- **🔍 智慧掃描破音字**：自動偵測可能的破音字。
- **✅ 即時校正面板**：提供直觀的面板進行人工校正。
- **☁️ 完全本地運算**：所有的處理都在 Google Apps Script 環境內完成，確保隱私。
